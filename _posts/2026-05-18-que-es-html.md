HTML significa HyperText Markup Language. Es el lenguaje que permite organizar
la estructura de una página web mediante etiquetas.

Con HTML se pueden crear títulos, párrafos, listas, enlaces, imágenes, formularios
y diferentes secciones dentro de un documento web.

Por ejemplo, una página sencilla puede tener un encabezado, un menú de navegación,
un contenido principal y un pie de página.
