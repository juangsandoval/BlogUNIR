CSS significa Cascading Style Sheets. Es el lenguaje que permite definir la
presentación visual de una página web.

Con CSS se pueden modificar colores, tamaños, márgenes, bordes, tipografías,
sombras y la distribución de los elementos en pantalla.

Mientras HTML organiza el contenido, CSS le da apariencia y mejora la experiencia
visual del usuario.
